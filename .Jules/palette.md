## 2026-02-04 - RTL Icon Positioning
**Learning:** In Arabic RTL interfaces, using logical CSS properties (e.g., `end-0`, `pe-10`) is essential for correct icon placement. `end-0` correctly places the icon on the left side of the input (the visual end in RTL), ensuring consistent design without RTL-specific overrides.
**Action:** Always use logical properties for spacing and positioning in this RTL-first codebase.
